# Backend
Accede a la carpeta del backend
   ```bash
   cd backend
Instala las dependencias del backend
    
    npm install

Ejecuta el backend
    
    npm start

El backend estará corriendo en http://localhost:3600.

----------------------------------------------------------
# Frontend (React)

    cd frontend

Instala las dependencias del frontend
    npm install

Ejecuta el frontend

    npm start
El frontend estará corriendo en http://localhost:4200.
